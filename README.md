## Requirements

- Python 2.7
- Frida Server
- ADB
- Rooted device

## Usage

Push the frida-server binary to the device, run it and run badada
 
Frida-server can be found under [releases](https://github.com/frida/frida/releases) page on Frida's Github repository.
